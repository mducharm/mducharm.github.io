# Proof Symbols

A simple static web app for typing mathematical proof notation using plain-text shortcuts. Type `\forall x \in S, P(x) -> Q(x)` and get `∀x ∈ S, P(x) → Q(x)`.

## Usage

Open `index.html` in a browser. No server, dependencies, or build step required.

- Type in the input area using LaTeX-style (`\forall`, `\in`) or informal (`->`, `!=`) shortcuts
- The preview updates live as you type
- Click **Copy** to copy the converted text to your clipboard
- Expand **Symbol Reference** to see all available triggers or click to insert
